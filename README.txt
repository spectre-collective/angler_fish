-> angler_fish is a demonstration of Transfer Learning using BERT pre-trained
   pre_trained model. Purposed with distinguishing fraudulent emails from 
   internal emails. uses nigerian fraud dataset from Kaggle as well as 
   enron dataset from the same site.

-> to run angler_fish , you must first set up a virtual environment

   navigate to terminal and enter the following command:
   
   -> python3 -m venv nameofyourchoosing

   -> source nameofyourchoosing/bin/activate

   -> pip3 install notebook

   -> jupyter notebook

 -> navigate to bertFish, click run, then pip3 install all of the missing packages
    until it works. will have set_.py file soon.
